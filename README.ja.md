# Font Compare (フォント比較ツール)

フォントを並べて比較するための、強力でクロスプラットフォームな Electron アプリケーション。
デザイナー、開発者、タイポグラフィ愛好家に最適です。

[English](README.md) | [简体中文](README.zh-CN.md) | [한국어](README.ko.md)

## ✨ 主な機能

- **並行比較**: 最大4つのフォントを同時にリアルタイムで比較できます。
- **キラー機能: 実際のレンダリングフォントインスペクタ**: Chrome DevTools Protocol (CDP) を利用して、テキストのレンダリングに実際に使用されているフォールバックフォントを正確に表示します。絵文字や文字が正しく表示されない理由を推測する必要はもうありません！
- **状態の永続化**: フォントの選択、テキスト入力、設定（サイズ、レイアウト、ズーム）は自動的に保存され、起動時に復元されます。
- **高度なタイポグラフィ制御**: フォントサイズ、行の高さ、文字間隔、フォントの太さ（太字）、スタイル（斜体）を調整します。
- **サブピクセルアンチエイリアス**: 最適化されたCSSにより、ギザギザのない鮮明でサブピクセルのClearTypeフォントレンダリングが保証されます。
- **グリッド/リストレイアウト**: 2x2グリッドまたは垂直リストレイアウトを切り替えます。

## 🚀 クイックスタート

### 前提条件
- Node.js (v16 以上)
- npm

### インストール

1. リポジトリのクローン:
   ```bash
   git clone https://github.com/fangsunjian/Font-Compare.git
   cd Font-Compare
   ```

2. 依存関係のインストール:
   ```bash
   npm install
   ```

3. アプリケーションの起動:
   ```bash
   npm start
   ```

## 🛠️ 使用技術
